# Photo Voting App

Это веб-приложение для проведения фотоконкурсов со "слепым" голосованием и полуавтоматическим сбором результатов.

## Ключевые возможности

- **5-звёздочная система рейтинга**: Участники оценивают фотографии по шкале от 1 до 5 звёзд.
- **Двойные лимиты**: Голосование ограничено как по количеству оценённых фотографий, так и по общей сумме звёзд.
- **Гибкая конфигурация**: Все ключевые параметры (пути к данным, лимиты, макет по умолчанию) настраиваются в файле `config.json`.
- **Настраиваемый вид**: Пользователи могут выбирать режим отображения галереи (сетка с разными пропорциями или оригинальные размеры).
- **Инструменты администратора**: Включает три утилиты:
    - `admin.html`: Для управления `config.json` и объединения результатов голосования.
    - `editor.html`: Для редактирования `photos.json` (вступительная статья, описания и порядок фото).
    - `prepare.html`: Для быстрой генерации `photos.json` из списка URL.

## Структура файлов

- `public/config.json` - Глобальные настройки приложения.
- `public/data/photos.json` - Данные о фотографиях и вступительная статья.
- `public/data/results.json` - Общие результаты голосования.
- `admin.html`, `admin.js` - Инструмент для администратора.
- `editor.html`, `editor.js` - Редактор данных о фотографиях.
- `prepare.html`, `prepare.js` - Инструмент для подготовки данных.

## Инструкция для администратора

### Шаг 1: Подготовка файлов

1.  **Сгенерируйте `photos.json`**:
    - Откройте `prepare.html` локально в браузере.
    - Вставьте список URL-адресов фотографий.
    - Нажмите "Сгенерировать" и скачайте `photos.json`.
    - Поместите его в папку `public/data`.

2.  **Отредактируйте `photos.json`**:
    - Откройте `editor.html` локально.
    - Он автоматически подгрузит `photos.json` из `public/data`.
    - Добавьте описания, вступительную статью и настройте порядок фотографий.
    - Скачайте обновлённый `photos.json` и замените им старый файл.

3.  **Настройте `config.json`**:
    - Откройте `admin.html` локально.
    - Загрузите `config.json` из папки `public`.
    - Установите лимиты голосования и макеты по умолчанию.
    - Скачайте и замените `config.json`.

4.  **Создайте `results.json`**:
    - В папке `public/data` создайте пустой файл `results.json` с содержимым `{}`.

5.  Соберите (`npm run build`) и выложите содержимое папки `dist` на сервер.

### Шаг 2: Сбор и объединение голосов

1.  Участники присылают вам JSON-объекты со своими голосами.
2.  Откройте на вашем сервере страницу `/admin`.
3.  Нажмите **«Шаг 1: Загрузить текущие результаты»**.
4.  Вставьте JSON от пользователя в поле **«Шаг 2...»**.
5.  Нажмите **«Шаг 3: Объединить результаты»**.
6.  Скопируйте или скачайте обновлённый JSON и замените `results.json` на сервере.
